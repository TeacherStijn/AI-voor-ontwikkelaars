# Q&A to ask questions on the documentation
## Pak je documentatie directory (out) in in een zipfile:
> Klik rechtermuis op de directory en kies voor 'inpakken'

## Voorzie de A.I. bot van achtergrond informatie:
> Start ChatGPT of een andere A.I.
>
> Sleep of upload op een andere manier de zipfile (of losse bestanden) aan

### Voorbeeldprompt indien je -geen- script gebruikt:
> Je bent een Q&A bot die vragen beantwoord over een applicatie die ik ga gebruiken waarvan ik de documentatie aan je lever via losse bestanden of een zipfile.
  Gebruik enkel deze documentatie om mijn vragen te beantwoorden. Vragen buiten de scope van de applicatie (en dus documentatie) mag je netjes afwijzen.
  Stel als retourvraag ook altijd de vraag "Is je vraag hiermee beantwoord?"
